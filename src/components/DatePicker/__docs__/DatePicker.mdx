# DatePicker

DatePicker component with different props.

#### Example

<Canvas of={DatePicker.Default} />

## Usage

```ts
import DatePicker from "./DatePicker";

const Example = () => {
  const handleDateChange = (date: string) => {
    console.log("Selected date:", date);
  };

  return (
    <DatePicker
      placeholder={"Select a date"}
      value={""}
      onChange={handleDateChange}
      minDate={"2023-01-01"}
      maxDate={"2023-12-31"}
    />
  );
};

export default Example;
```

#### Arguments

- **placeholder** string - A string that represents the placeholder text of the input field.
- **value** string - A string representing the current value of the date picker. It should be in the format "YYYY-MM-DD".
- **onChange** (date: string) => void - A function that is called when the date is changed. It receives a string representing the selected date.
- **minDate** string - A string representing the minimum selectable date in the format "YYYY-MM-DD".
- **maxDate** string - A string representing the maximum selectable date in the format "YYYY-MM-DD".
